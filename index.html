Dziedzina i miejsca zerowe funkcji

Funkcja: f(x) = 2x² + 2x - 6

1. Dziedzina

Funkcja kwadratowa to wielomian, który jest określony dla wszystkich liczb rzeczywistych.

Dziedzina: Df = ℝ

2. Współczynniki

Dla funkcji f(x) = 2x² + 2x - 6:

  • a = 2 (współczynnik przy x²)
  • b = 2 (współczynnik przy x)
  • c = -6 (wyraz wolny)

3. Obliczanie miejsc zerowych

Aby znaleźć miejsca zerowe, rozwiązujemy równanie f(x) = 0, czyli:

x² - x - 6 = 0

Stosujemy wzór kwadratowy:

x = (-b ± √Δ) / 2a

Gdzie Δ to delta, obliczana jako:

Δ = b² - 4ac

Po podstawieniu odpowiednich wartości można wyznaczyć pierwiastki funkcji.

Podsumowanie

  • Dziedzina: Df = ℝ
  • Miejsca zerowe: Obliczone na podstawie delty.
