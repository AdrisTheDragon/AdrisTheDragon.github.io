Równania kwadratowe i delta

Równanie kwadratowe w postaci ogólnej to:

ax² + bx + c = 0

Wyróżnik (delta) jest obliczany według wzoru:

Δ = b² - 4ac

Przypadki w zależności od wartości delty:

1. Δ > 0:

x₁ = (-b - √Δ) / (2a)

x₂ = (-b + √Δ) / (2a)

2. Δ = 0:

x = -b / (2a)

3. Δ < 0:

Brak rozwiązań.

Rozwiązanie równania kwadratowego f(x) = x² - x - 6

Aby rozwiązać równanie kwadratowe f(x) = x² - x - 6, znajdź miejsca zerowe funkcji.

Krok 1: Znajdź współczynniki

Równanie ma postać ogólną ax² + bx + c = 0, gdzie:

  • a = 1
  • b = -1
  • c = -6

Krok 2: Wyznacz deltę

Delta Δ to:

Δ = b² - 4ac
Δ = (-1)² - 4(1)(-6) = 1 + 24 = 25

Krok 3: Znajdź pierwiastki równania

Pierwiastki wyznacza się ze wzoru kwadratowego:

x = (-b ± √Δ) / 2a
x = (1 ± √25) / 2(1)

Rozwiązując:

x₁ = (1 + 5) / 2 = 3
x₂ = (1 - 5) / 2 = -2

Wynik:

Miejsca zerowe funkcji to:

  • x₁ = 3
  • x₂ = -2

Rozwiązanie zadania

Rozwiążmy wyrażenie f(x) = x³ - 3x² + 3x - 1 przy użyciu wzoru skróconego mnożenia:

Chcemy wyrazić funkcję f(x) w postaci (a - b)³ = a³ - 3a²b + 3ab² - b³.

Krok 1: Dopasowanie funkcji do wzoru

Funkcja f(x) = x³ - 3x² + 3x - 1 jest w postaci, którą możemy zapisać jako (x - 1)³.

Krok 2: Sprawdzenie obliczeń

Sprawdzamy, czy rozkład (x - 1)³ daje nam wyrażenie x³ - 3x² + 3x - 1:

(x - 1)³ = x³ - 3x² + 3x - 1

Rzeczywiście, mamy: f(x) = (x - 1)³.
