Zadanie 1a

Funkcja: f(x) = x^2 - x - 6

1. Wyznaczenie dziedziny funkcji:

Funkcja f(x) = x^2 - x - 6 jest wielomianem, a wielomiany są określone dla wszystkich liczb rzeczywistych.

Dziedzina: ℝ

2. Wyznaczenie miejsc zerowych funkcji:

Aby znaleźć miejsca zerowe funkcji, rozwiązujemy równanie:

f(x) = 0, czyli x^2 - x - 6 = 0

Krok 1: Obliczenie delty (Δ)

Wzór na deltę: Δ = b^2 - 4ac

Współczynniki równania to:

  • a = 1
  • b = -1
  • c = -6

Obliczenia:

Δ = (-1)^2 - 4 * 1 * (-6) = 1 + 24 = 25

Krok 2: Obliczenie pierwiastków równania

Pierwiastki równania kwadratowego obliczamy wzorem:

x₁, x₂ = (-b ± √Δ) / (2a)

Podstawiając wartości:

x₁, x₂ = (1 ± √25) / 2

x₁ = (1 - 5) / 2 = -2

x₂ = (1 + 5) / 2 = 3

Podsumowanie:

  • Dziedzina: ℝ
  • Miejsca zerowe: x = -2 oraz x = 3

Zadanie 1b

Wyznaczyć dziedzinę i miejsca zerowe funkcji:

f(x) = x³ - 3x² + 3x - 1

Rozwiązanie

1. Dziedzina

Funkcja f(x) to wielomian, który jest określony dla każdej wartości x.

Dziedzina: R (zbiór liczb rzeczywistych).

2. Miejsca zerowe

Rozwiązujemy równanie:

f(x) = 0

Czyli:

x³ - 3x² + 3x - 1 = 0

Przekształcenie za pomocą wzoru skróconego mnożenia

Funkcja jest w postaci wzoru skróconego mnożenia:

(x - 1)³ = x³ - 3x² + 3x - 1

Zatem:

f(x) = (x - 1)³

Aby znaleźć miejsca zerowe:

(x - 1)³ = 0

Stąd:

x - 1 = 0 x = 1

Wyniki

  • Dziedzina: R
  • Miejsce zerowe: x = 1 (o trzykrotnej krotności)

Funkcja dotyka osi x w punkcie x = 1, ale jej nie przecina.

Zadanie 1c

Funkcja: f(x) = 1 / (x² - 9)

1. Dziedzina funkcji

Funkcja f(x) jest określona, gdy mianownik x^2 - 9 ≠ 0, ponieważ dzielenie przez zero jest niedozwolone.

Rozwiązujemy równanie x^2 - 9 = 0:

        x² - 9 = 0
        x² = 9
        x = ±3

Zatem x = 3 i x = -3 są punktami, w których mianownik wynosi zero. Wykluczamy te punkty z dziedziny.

Dziedzina funkcji to: Df = R \ {-3, 3}.

2. Miejsca zerowe funkcji

Miejsca zerowe funkcji to takie wartości x, dla których f(x) = 0. 

        f(x) = 0
        1 / (x² - 9) = 0

Równanie 1 / (x² - 9) = 0 nie ma rozwiązania, ponieważ licznik ułamka wynosi 1, więc wyrażenie nigdy nie przyjmuje wartości 0.

Zatem funkcja nie ma miejsc zerowych.

Podsumowanie

  • Dziedzina: Df = R \ {-3, 3}
  • Miejsca zerowe: brak

Zadanie 1d

Analiza funkcji: f(x) = ln(√x - 4)

1. Dziedzina funkcji

Funkcja jest określona tylko wtedy, gdy argument logarytmu jest większy od zera. Zatem musimy zapewnić, że:

√x - 4 > 0

Rozwiązujemy nierówność:

√x > 4

Teraz podnosimy obie strony nierówności do kwadratu (pamiętając, że pierwiastek jest liczbą nieujemną):

x > 16

Dziedzina funkcji to:

D(f) = (16, ∞)

2. Miejsca zerowe funkcji

Aby znaleźć miejsca zerowe, rozwiązujemy równanie:

f(x) = ln(√x - 4) = 0

Logarytm naturalny z liczby 1 wynosi 0, więc:

√x - 4 = 1

Dodajemy 4 do obu stron:

√x = 5

Teraz podnosimy obie strony do kwadratu:

x = 25

Miejscem zerowym funkcji jest:

x = 25

Podsumowanie

Dziedzina funkcji: D(f) = (16, ∞)

Miejsce zerowe funkcji: x = 25

Zadanie 1e

Funkcja

Funkcja ma postać: f(x) = √(x + 2) - 1/√(4 - x)

1. Dziedzina funkcji

Funkcja jest określona, gdy:

  • √(x + 2) jest określona, gdy x + 2 ≥ 0, czyli x ≥ -2
  • 1/√(4 - x) jest określona, gdy 4 - x > 0, czyli x < 4

Stąd dziedzina funkcji to zbiór: x ∈ [-2, 4)

2. Miejsca zerowe funkcji

Aby znaleźć miejsca zerowe, rozwiązujemy równanie:

√(x + 2) - 1/√(4 - x) = 0

Przekształcamy je do postaci:

√(x + 2) = 1/√(4 - x)

Podnosimy obie strony do kwadratu:

x + 2 = 1 / (4 - x)

Mnożymy obie strony przez (4 - x):

(x + 2)(4 - x) = 1

Rozwijamy i upraszczamy:

4x - x² + 8 - 2x = 1 -x² + 2x + 8 = 1

Przenosimy wszystko na jedną stronę:

-x² + 2x + 7 = 0

Mnożymy przez -1:

x² - 2x - 7 = 0

Obliczenie delty

Obliczamy deltę:

Δ = b² - 4ac

Dla równania x² - 2x - 7 = 0, mamy:

  • a = 1
  • b = -2
  • c = -7

Podstawiamy do wzoru na deltę:

Δ = (-2)² - 4(1)(-7) = 4 + 28 = 32

Obliczenie pierwiastków

Podstawiamy deltę do wzoru na pierwiastki równania kwadratowego:

x = (-b ± √Δ) / 2a

Podstawiamy wartości:

x = (2 ± √32) / 2

Upraszczamy:

x = (2 ± 4√2) / 2

Po podzieleniu przez 2:

x = 1 ± 2√2

Przybliżone miejsca zerowe

Obliczamy przybliżoną wartość 2√2:

2√2 ≈ 2 × 1.414 ≈ 2.828

Stąd:

x₁ ≈ 1 + 2.828 = 3.828 x₂ ≈ 1 - 2.828 = -1.828

Podsumowanie

Miejsca zerowe funkcji to:

  • x₁ ≈ 3.828
  • x₂ ≈ -1.828
