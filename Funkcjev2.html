Dziedzina i miejsca zerowe

Krok po kroku dla f(x) = x² - x - 6

1. Dziedzina funkcji

Funkcja f(x) = x² - x - 6 to funkcja kwadratowa, która jest wielomianem.

Wielomiany są określone dla wszystkich liczb rzeczywistych (ℝ).

Dziedzina: Df = ℝ

2. Miejsca zerowe funkcji

Aby znaleźć miejsca zerowe, należy rozwiązać równanie:

f(x) = 0

czyli:

x² - x - 6 = 0

Krok 1: Rozwiąż równanie kwadratowe

Zapisz równanie w postaci ogólnej: ax² + bx + c = 0, gdzie:

  • a = 1
  • b = -1
  • c = -6

Krok 2: Oblicz deltę (Δ) ze wzoru:

Δ = b² - 4ac

Podstaw wartości:

Δ = (-1)² - 4(1)(-6) = 1 + 24 = 25

Krok 3: Oblicz pierwiastki równania

Pierwiastki równania kwadratowego wyznaczamy ze wzoru:

x1,2 = (-b ± √Δ) / 2a

Podstaw wartości:

x1,2 = (-(−1) ± √25) / 2(1) = (1 ± 5) / 2

Oblicz:

  • x1 = (1 − 5) / 2 = -4 / 2 = -2
  • x2 = (1 + 5) / 2 = 6 / 2 = 3

Podsumowanie:

  • Dziedzina: Df = ℝ
  • Miejsca zerowe: x = -2 oraz x = 3
